# 🖥️ ABO Sim — Versão Portátil (.exe)

## Como usar

1. Baixe o arquivo **`ABO-Sim-v1.0.0.exe`**
2. Dê **duplo clique** no arquivo
3. Pronto! O app abre em uma janela própria (sem precisar de navegador)

## O que é?

Esta é uma versão **portátil** do ABO Sim. Não precisa instalar nem extrair — o programa roda direto do arquivo `.exe`. É como aqueles programas "portáteis" que você conhece.

## Requisitos

- Windows 10 ou 11 (64-bit)
- Não precisa de navegador instalado
- Não precisa de internet
- Não precisa de permissão de administrador

## Como funciona?

O arquivo contém tudo que o app precisa (HTML, CSS, JavaScript, sons e o próprio navegador embutido). É 100% offline e independente.

## Tamanho

~67 MB (inclui o Chromium embutido para rodar sem depender do navegador do computador).

## Diferenças da versão web

- Abre em janela própria (como um app de verdade)
- Não precisa de Chrome, Edge, Firefox, etc.
- Funciona em computadores sem navegador instalado
- Ícone na barra de tarefas e menu iniciar
- Pode ser copiado para pen drive e rodar em qualquer PC
